# Quickstart For Staff Users

Audience: regular store staff using the portal after the admin has created your login.

## Log In

1. Open the portal website.
2. Click `Store staff login`.
3. Enter your username and password.
4. If the portal asks you to change a temporary password, do that first.

## Find Existing Work

Use the portal home cards:

- `Contracts`: search by customer name, phone, address, invoice, estimate, or record ID.
- `Estimates`: search saved estimates.
- `Print`: open blank forms when paper is needed.

Search before creating a new record so duplicate customers and duplicate contracts are easier to avoid.

## Create A Contract

1. Go to `Contracts`.
2. Search for the customer first.
3. Choose the existing customer if found, or click `Add New`.
4. Fill customer details.
5. Move through the contract workflow.
6. Save as you go.
7. Generate/send only after the required sections are complete.

If the portal warns about a possible duplicate, compare the record ID and customer details before creating another version.

## Create Or Use An Estimate

1. Search existing estimates first.
2. If an estimate exists, use it with the contract.
3. If not, create a new estimate from the estimate area.
4. Customer name, address, and phone are required for estimates.
5. Save the estimate before returning to the contract.

## Send Or View Documents

From a contract record, use the record actions:

- View details.
- Open signing.
- Email link.
- Print PDF.
- Open signable PDF.
- Edit draft.

Use `View` for records and documents. Use `Show` for revealing passwords or expanded fields.

## Customer Signing

Customers can use the signing link or customer portal. The signed packet saves in the portal after completion.

The first-contract password should not be written in customer-facing email text. Staff can use the portal to open the signing link and documents.

## Exit

Use `Exit` when done, especially on shared computers.
